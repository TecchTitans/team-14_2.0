Libraries:
Main game engine
LibGDX - https://libgdx.com/
JSON Library - https://github.com/stleary/JSON-java

Tools:
For compiling UI features together I used LibGDX's skin composer to generate an atlas file of all the components
https://libgdx.com/wiki/tools/skin-composer

For compiling textures I used GDX Texture Packer, which just takes pre-made sprites and merges them into one image
https://libgdx.com/wiki/tools/texture-packer

For making maps:
https://thorbjorn.itch.io/tiled

Player sprite (Male) - https://cuddle-bug.itch.io/apocalypse (recoloured)
